ToolBelt

App Description: iOS app that allows users to share power tools within their community, similar to AirBnB. Users register with Facebook OAuth and set their own location, and manage their list of tools within the app. Users can search for tools near their current location via keyword, and can view the results in a list or map view. To borrow or loan a tool, they can chat directly with the tool's owner to set the meetup. They can rate their interactions with an owner, and all owners' ratings are viewable by other users. Users can also flag other users in the event of damage or theft, which will prompt an internal review of the flagged user.

Technologies Used: Swift, XCode, Ruby on Rails, PostGreSQL, Firebase, Alamofire API client, Facebook OAuth, Heroku

  • Date: May 2015
  • Service: Web Development

PB&J Shop

App Description: Web application that emulates a modern day online store. It allows a user to create, login, and logout. Once a user creates an account an Email will be sent to them. Users are able place items into their cart in bulk. After purchasing the items, users are able to view their prior orders.

Technologies Used: Ruby on Rails, PostGreSQL, Redis, ActionMailer, Heroku

  • Service: Web Development
